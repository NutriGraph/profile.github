The Global Food Intelligence API for Modern Food Applications

Welcome to the official GitHub organization for NutriGraph. Here you will find the tools, documentation, and code samples you need to integrate our powerful food intelligence API into your applications.

NutriGraph provides programmatic access to a database of over 10 million food products, complete with high-resolution nutrition data, allergen sensitivities, ingredient analysis, and more.
